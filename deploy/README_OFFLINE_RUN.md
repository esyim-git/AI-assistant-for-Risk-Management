# Offline Run Guide

## 실행 방법

1. ZIP 파일의 SHA256 해시를 확인합니다.
2. 승인된 폴더에 압축을 풉니다.
3. `run.bat` 또는 `RiskManagementAI.exe`를 실행합니다.
4. Dashboard에서 Offline Mode 상태를 확인합니다.
5. 샘플 데이터 분석으로 정상 실행을 확인합니다.

## 필요사항

- Windows 11
- Excel 2021

## 필요하지 않은 것

- Visual Studio
- VS Code
- .NET SDK
- Python
- Git
- 인터넷

## 주의

초기 MVP에서 Local LLM 모델이 없으면 AI 생성 기능은 제한됩니다.
룰 검사, 데이터 프로파일링, 샘플 분석 기능은 사용할 수 있습니다.
