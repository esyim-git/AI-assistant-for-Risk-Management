# Knowledge Base Ingestion Policy

## 원칙

모델에 직접 재학습시키지 않고, 승인된 문서를 검색형 Knowledge Base로 활용한다.

## 문서 분류

| 분류 | 예시 | 저장 방식 |
|---|---|---|
| PUBLIC_REG | 금융투자업규정, 시행세칙 | 공개 규정 KB |
| NCR_GUIDE | NCR 산정기준 해설 | 회사 공식본 승인 후 적재 |
| INTERNAL_RULE | 내부규정 | 권한통제형 RAG |
| RISK_THEORY | VaR, ES, DV01 등 | 자체 요약 카드 |
| PAPER | 논문 | 라이선스 확인 후 요약 중심 |
| FRM_SUMMARY | FRM 개념 요약 | 원문 적재 금지, 자체 요약 |

## 내부규정 처리

내부규정은 집/공개 Repository에 포함하지 않는다.
회사 운영환경에서 문서오너 승인 후 다음 정보만 우선 저장한다.

- 문서명
- 버전
- 시행일
- 조항 번호
- 요약
- 업무 적용 포인트
- 접근권한
- 원문 위치 참조

## 답변 시 필수 표시

- 문서명
- 버전
- 시행일
- 조항/문단
- 적용 기준일
- 확인 필요사항
