# Project Vision

## 목적

리스크관리 업무에서 반복되는 SQL 작성, Excel 2021 리포트 작성, 데이터 품질 점검, 한도 모니터링, NCR/규정 검색 업무를 Local AI Assistant로 표준화한다.

## 핵심 메시지

> AI가 자동으로 실행하는 도구가 아니라, 사람이 검토 가능한 초안·검증·근거·보고서를 제공하는 리스크관리 Copilot이다.

## 대상 업무

- 시장리스크 모니터링
- 한도 사용률 점검
- Valuation/Validation 보조
- Hedge Risk 분석
- 유동성/만기 Gap 분석
- NCR 산정 검증 보조
- 규정/NCR 문서 검색
- SQL/VBA 작성 보조

## 성공 기준

- SQL 작성 및 검증 시간 단축
- Excel 리포트 작성 시간 단축
- 기준일/중복/Null/한도 미매핑 등 오류 탐지 증가
- 팀 공통 템플릿화
- 작업 이력과 검증 근거 보존
