# PRD

## 제품명

Risk Management AI Assistant

## 사용자

- 리스크관리 담당자
- OTC/파생상품 모니터링 담당자
- NCR 산출/검증 담당자
- 보고서 작성 담당자

## 주요 기능

### MVP-1

- SQL Safety Checker
- VBA Safety Checker
- Excel 2021 Function Checker
- 작업 로그 구조
- 피드백 로그 구조

### MVP-2

- SQL/VBA Prompt Template Engine
- Golden6 Export 데이터 프로파일링
- Risk Column Mapper

### MVP-3

- Excel 2021 리포트 생성
- Limit Monitoring 대시보드
- Exception List 생성

### MVP-4

- Regulation/NCR KB 구조
- 공개 규정 카탈로그
- 내부규정 placeholder

### MVP-5

- WPF UI 통합
- Local Model Pack 연동 옵션
- 팀 승인 예제 검색

## 제외 범위

초기 범위에서 제외한다.

- Golden6 자동 접속
- 운영 DB 자동 실행
- VBA 자동 실행
- 내부규정 원문 저장
- 회사 실데이터 학습
- 모델 가중치 자동 재학습
- 외부 API 호출
