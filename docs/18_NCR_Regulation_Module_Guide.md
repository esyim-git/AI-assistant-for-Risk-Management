# NCR Regulation Module Guide

## 목적

NCR 관련 규정/해설/내부기준을 기준일과 산식 버전 관점에서 검색하고, 검증 SQL/체크리스트를 생성한다.

## 답변 포맷

1. 질문 요약
2. 적용 기준일
3. 참조 문서/규정 버전
4. 관련 조항/내부기준
5. 업무 적용 판단
6. 필요 데이터
7. 검증 SQL
8. 리스크관리 주의사항
9. 승인 필요사항

## Hidden Risk

- 구버전 산식 사용
- 기준일 불일치
- 상품군 매핑 오류
- 헤지 인정 조건 오판
- 내부규정 미반영
- 수기조정 누락
