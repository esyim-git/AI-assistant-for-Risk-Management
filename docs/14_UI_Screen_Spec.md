# UI Screen Spec

## 기본 레이아웃

```text
좌측 메뉴
- Dashboard
- SQL Assistant
- VBA Assistant
- Data Analyzer
- Risk Dashboard
- Regulation/NCR Search
- Feedback Center
- History
- Settings

중앙
- 작업 입력/결과

우측
- Safety Check
- Assumptions
- Validation
- Hidden Risk
```

## Dashboard

- 오늘 작업 수
- 위험 코드 탐지 수
- 분석 데이터셋 수
- 생성 리포트 수
- Local Model 상태
- Offline Mode 상태

## SQL Assistant

- 업무 요청 입력
- SQL 초안
- 검증 SQL
- 위험 검사 결과
- 복사 버튼

## VBA Assistant

- 업무 요청 입력
- VBA 초안
- Excel 2021 호환성 검사
- 위험 API 검사

## Data Analyzer

- CSV/XLSX 파일 선택
- 데이터 프로파일
- 컬럼 매핑
- 예외 리스트

## Regulation/NCR Search

초기에는 placeholder 화면으로 구현한다.
