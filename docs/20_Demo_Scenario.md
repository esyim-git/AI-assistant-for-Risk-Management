# Demo Scenario

## 데모 목표

더미 데이터 기준으로 리스크관리 Assistant의 업무 흐름을 보여준다.

## 시나리오 1: SQL 생성/검증

1. 사용자 요청 입력
2. SQL 초안 생성
3. 금지 명령 검사
4. 검증 SQL 표시
5. Hidden Risk 표시

## 시나리오 2: Golden6 Export 분석

1. 더미 CSV 업로드
2. 행 수/컬럼 수 표시
3. Null/중복 점검
4. 한도 사용률 Top N 표시
5. Exception List 생성

## 시나리오 3: Excel 2021 리포트

1. Summary 시트
2. DATA_PROFILE 시트
3. LIMIT_MONITORING 시트
4. EXCEPTION_LIST 시트
5. SQL_USED/CHANGE_LOG 시트
